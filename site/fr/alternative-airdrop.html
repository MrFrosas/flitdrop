La réponse claire

AirDrop est réservé à Apple. Cette alternative à AirDrop, non.

AirDrop est excellent, mais il reste réservé aux appareils Apple. Dès qu'un PC Windows, un téléphone Android ou une machine Linux entre dans la boucle, tout s'arrête. Flitdrop est l'alternative à AirDrop gratuite et multiplateforme pensée pour ça : elle envoie vos photos en pleine qualité, vos fichiers, vos dossiers et même votre presse-papiers directement entre deux appareils, iPhone vers Windows, Android vers Mac, Samsung vers Linux, dans tous les sens, sur votre propre Wi-Fi et jamais par le cloud.

Pourquoi une alternative

Pourquoi AirDrop a besoin d'une alternative (et ce que Flitdrop corrige)

  1. 1

    AirDrop, uniquement entre appareils Apple

    Il n'existe pas d'AirDrop officiel pour Windows, Android ou Linux, ce qui bloque les foyers et bureaux mixtes.

  2. 2

    Flitdrop couvre toutes les combinaisons

    iPhone, Android et Samsung d'un côté ; Windows, Mac et Linux de l'autre ; n'importe quel duo fonctionne.

  3. 3

    Pleine qualité, sans attendre

    Le cloud et les mails compressent vos photos ou vous font attendre un envoi ; Flitdrop transmet le fichier original en pleine qualité.

  4. 4

    Rien à installer sur le téléphone

    La plupart des alternatives exigent une appli sur les deux appareils ; avec Flitdrop, le téléphone n'a besoin que de son navigateur.

Comment ça marche

Comment Flitdrop fonctionne : un scan, puis vous envoyez tout

Installez sur l'ordinateur, liez le téléphone en un seul scan, puis glissez vos fichiers. Voilà tout le parcours.

  1. 1

    Installez sur votre ordinateur

    Installez l'appli gratuite sur votre ordinateur (Windows, Mac ou Linux) ; elle se met à jour toute seule, aucune version à traquer.

  2. 2

    Scannez le QR code une fois

    Scannez le QR code une fois depuis votre téléphone pour l'appairer ; rien à installer sur le téléphone, il ouvre simplement une web-app légère dans son navigateur.

  3. 3

    Glissez, ça arrive

    Glissez un fichier, un dossier entier ou un lot de photos et il arrive sur l'autre appareil en qualité d'origine, directement sur votre Wi-Fi, sans compte ni inscription.

Transparence totale

Ce que Flitdrop ne sait pas faire (encore)

Flitdrop est jeune et gratuit, et on préfère vous signaler les points faibles tout de suite plutôt que de vous les faire découvrir en plein transfert. Voici précisément ce qu'il ne fait pas encore, et pourquoi.

  • Au tout premier lancement, SmartScreen sous Windows ou Gatekeeper sous macOS vous avertira au sujet d'un « développeur non identifié ». L'application est sûre : cet avertissement veut simplement dire qu'on n'a pas encore payé de certificat de signature. Cliquez sur « Informations complémentaires » puis « Exécuter quand même » sous Windows, ou faites un clic droit puis Ouvrir sous Mac ; le code étant public, vous pouvez le vérifier vous-même.

  • Il n'y a pas d'application native pour votre téléphone. Rien ne s'installe sur le téléphone, et c'est voulu : vous scannez un QR code une fois et tout se passe dans votre navigateur. En contrepartie, tout tourne dans une page web plutôt que dans une application dédiée.

  • Les appareils ne se trouvent pas encore tout seuls, donc vous scannez un QR code pour établir la première connexion. La détection automatique sur le réseau local est prévue ; pour l'instant, le QR est la petite étape qui vous met en relation.

  • Votre téléphone ne peut pas recevoir de fichier quand l'écran est verrouillé ou le navigateur en arrière-plan. Comme le côté téléphone est une page web et non une application de fond, gardez l'onglet Flitdrop ouvert et l'écran allumé pendant un transfert.

  • Les deux appareils doivent être sur le même réseau Wi-Fi. Flitdrop envoie en pair à pair sur votre réseau local, pas à travers Internet, et c'est ce qui le garde rapide et privé. Sans Wi-Fi commun, votre ordinateur peut créer un point d'accès pour que les deux se rejoignent quand même.

  • Il n'existe aujourd'hui que des versions Windows et macOS, donc Linux n'a pas encore de version prête à télécharger. Une version Linux est prévue, et d'ici là le code source est public si vous voulez la compiler vous-même.

  • Flitdrop est à source disponible sous licence FSL-1.1-ALv2, pas open source au sens classique. Le code est public et entièrement vérifiable, mais la licence interdit de le revendre ou d'en faire un produit concurrent ; chaque version passe sous Apache 2.0 au bout de deux ans. On préfère le dire clairement plutôt que de laisser croire à du MIT.

Alternative à AirDrop : questions fréquentes

Existe-t-il un AirDrop pour Windows et Android ?

Pas officiellement : AirDrop ne fonctionne qu'entre appareils Apple. Flitdrop comble ce manque en tant qu'alternative multiplateforme reliant iPhone, Android et Samsung avec Windows, Mac et Linux, dans tous les sens.

Dois-je installer une appli sur mon téléphone ?

Non. Vous installez l'appli uniquement sur l'ordinateur, puis vous scannez un QR code une fois pour l'appairer. Le téléphone ouvre simplement une web-app légère dans son navigateur, rien à télécharger sur un store.

Flitdrop est-il gratuit ?

Oui. Flitdrop est gratuit pour envoyer des fichiers sur votre propre réseau. Les transferts se font directement entre vos appareils, donc aucun coût de serveur ni abonnement pour les envois locaux.

Envoie-t-il les photos en pleine qualité sans passer par le cloud ?

Oui. Flitdrop transfère le fichier d'origine en pleine qualité, directement d'appareil à appareil sur votre Wi-Fi local. Ce n'est pas un service cloud, donc rien n'est envoyé ni compressé par un serveur tiers.

Est-ce sécurisé et faut-il un compte ?

Chaque transfert est chiffré de bout en bout et il n'y a aucun compte à créer. On appaire avec un QR code et on envoie ; vos fichiers restent entre vos deux appareils.

Puis-je transférer d'un iPhone vers un PC Windows ?

Oui, c'est l'une des raisons d'être de Flitdrop. Installez l'appli sur votre PC Windows, scannez le QR code depuis votre iPhone, et envoyez photos, fichiers ou texte du presse-papiers dans les deux sens sur votre Wi-Fi.
